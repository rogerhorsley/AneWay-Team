# AneWay 团队知识库

共享知识、复盘、技能沉淀的团队协作仓库。

## 团队成员

| 成员 | 职责 | 目录 |
|------|------|------|
| team-lead | 团队负责人，统筹协调 | [team-lead/](./team-lead/) |
| product-manager | 产品规划、需求分析 | [product-manager/](./product-manager/) |
| user-researcher | 用户调研、行为分析 | [user-researcher/](./user-researcher/) |
| designer | UI/UX 设计 | [designer/](./designer/) |
| rd-engineer | 技术开发、架构设计 | [rd-engineer/](./rd-engineer/) |
| marketing | 市场推广、品牌运营 | [marketing/](./marketing/) |
| compliance | 合规审查、风险控制 | [compliance/](./compliance/) |
| strategy | 战略规划、市场分析 | [strategy/](./strategy/) |

## 目录结构

每位成员拥有三个子目录：

```
member-name/
├── knowledge/      # 专业知识沉淀
├── retrospectives/ # 复盘与反思
└── skills/         # 技能学习记录
```

## 团队制度

- 每日 19:00 发送今日简报到飞书群
- 每日复盘，持续自我迭代
- 知识共享，共同成长
